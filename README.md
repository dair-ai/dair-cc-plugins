# DAIR.AI Claude Code Plugins

A collection of Claude Code plugins for building AI-powered applications.

## Installation

Add the marketplace to Claude Code:

```bash
/plugin marketplace add dair-ai/dair-cc-plugins
```

List available plugins:

```bash
/plugin
```

Install a plugin:

```bash
/plugin install <plugin-name>@dair-cc-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [ai-agent-builder](./ai-agent-builder) | Build modern web apps with Next.js, shadcn/ui, Supabase, and Claude AI |
